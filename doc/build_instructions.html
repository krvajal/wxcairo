Directory Structure

The projects and makefiles are all setup assuming the following directory structure:

   3rdparty/wxwidgets
   3rdparty/cairo
   wxcairo/...

Building Cairo

Building on a Mac

I had to install the following packages via Fink:

   fink install freetype219
   fink install fontconfig2-dev

Then I compiled Cairo using the following options

   ./configure --disable-xlib --enable-quartz=yes --enable-atsui=yes --enable-freetype=no
   make

Building wxWidgets

On windows I usually build wxWidgets using the following commands:

   vsvars32
   cd 3rdparty\build\msw\
   nmake /f makefile.vc MONOLITHIC=1 SHARED=0
